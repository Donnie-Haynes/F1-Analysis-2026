# F1-Analysis-2026

# 🏎️ F1 2026 Data Analysis Project

A real-time Formula 1 data analysis project tracking the 2026 season using Python, FastF1, and Pandas. Built as a portfolio project combining data analytics and secure data pipeline practices.

---

## 📊 Project Overview

This project pulls live F1 race data and analyzes it across three focus areas:

- **Season Performance Tracker** — Championship standings, points progression, and driver trends race by race
- **Pit Stop Strategy Analyzer** — Tire compounds, pit timing, and how strategy impacts race outcomes
- **Secure Data Pipeline** — API data fetching with logging, validation, and error handling best practices

---

## 🛠️ Tools & Technologies

- Python 3.9
- FastF1 — F1 telemetry and race data API
- Pandas — Data manipulation and analysis
- Matplotlib — Data visualization
- Jupyter Notebook — Interactive development
- Git & GitHub — Version control
